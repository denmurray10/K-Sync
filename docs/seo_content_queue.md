## Weekly Editorial Rules

- Treat the article title, H1, and opening paragraph as the primary SEO surface.
- Keep each article tied to one core phrase and one clear user intent.
- Run the internal-link pass after every article publish.
- Prefer articles that can be refreshed weekly or monthly over one-off disposable posts.

## Measurement Notes

- If a page earns impressions but low CTR, adjust title and description before replacing it.
- If a page gets clicks but weak depth, improve internal linking and CTA clarity before publishing a sibling page.
- Duplicate the pattern of the fastest-impression page first rather than inventing a new structure every week.
