# Day 3 Internal Linking Checklist

Use this to keep the core SEO pages connected as topic clusters.

## Homepage

- Link to `live`
- Link to `charts`
- Link to `comebacks`
- Link to `news`
- Link to at least 3 SEO destination pages

## Listen Free

- Link back to `home`
- Link to `live`
- Link to at least 3 SEO destination pages
- Keep one route focused on free listening, not general discovery

## News / Blog

- Link to `discover_new_kpop_music`
- Link to `best_kpop_playlist_2026`
- Link to one geo or mood destination page
- Keep using the blog internal link pass after article generation

## Comebacks

- Link to `live`
- Link to `charts`
- Link to `idols`
- Link to `discover_new_kpop_music`

## Landing Pages

- Every landing page should link back into:
  `live`, `charts`, `news` or `blog_page`
- Playlist and discovery pages should also link into `comebacks`

## Rules

- Use descriptive anchor text
- Never link to lab, test, preview, or placeholder routes
- Do not rename route names or slugs to chase SEO
