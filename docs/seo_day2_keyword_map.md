# Day 2 Keyword Map

This map keeps the core discovery pages from overlapping in search.

## Core Page Ownership

- `/`
  Primary keyword cluster: `kpop radio`, `kpop radio online`, `kpop radio uk`
  Intent: broad station discovery
  Primary CTA: listen live

- `/listen-free/`
  Primary keyword cluster: `free kpop radio`, `kpop music free online no download`, `listen to kpop online free`
  Intent: free listening with low-friction entry
  Primary CTA: play live free

- `/live/`
  Primary keyword cluster: `live kpop stream`, `kpop live radio`, `kpop radio live now`
  Intent: real-time now-playing and live station engagement
  Primary CTA: keep listening

- `/news/`
  Primary keyword cluster: `kpop news`, `discover new kpop music`, `kpop comeback updates`
  Intent: editorial discovery and freshness
  Primary CTA: read latest story

- `/idols/`
  Primary keyword cluster: `kpop idols`, `kpop groups`, `kpop artists`
  Intent: artist discovery
  Primary CTA: open artist page

- `/schedule/`
  Primary keyword cluster: `kpop radio schedule`, `live kpop shows`, `kpop radio timetable`
  Intent: programming lookup
  Primary CTA: tune in to the next show

- `/comebacks/`
  Primary keyword cluster: `kpop comebacks`, `upcoming kpop releases`, `kpop comeback calendar`
  Intent: release tracking and anticipation
  Primary CTA: follow upcoming release

## Rules

- Keep one clear H1 per page that matches the intent above.
- Avoid reusing the same lead paragraph across `home`, `listen-free`, and `live`.
- Keep `home` broad, `listen-free` frictionless, and `live` real-time.
- Route changes are off-limits because analytics and internal links depend on stable names.
