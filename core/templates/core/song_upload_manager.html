StrayKids Bucket Uploader

Upload A Folder Of Songs

Select a folder of `.mp3`, `.wav`, `.m4a`, or `.flac` files. Each successful upload goes into the configured Backblaze StrayKids/Music path and is synced into the radio library so it can be used by the live player. Embedded cover art is extracted from each song, uploaded to the shared radio artwork store, and saved onto the track for reuse across the site. Any failures are logged to tmp/song_upload_failures_latest.json for retry later.

What Happens

  • Each audio file is uploaded into StrayKids/Music/... in Backblaze.
  • Live radio accepts files from StrayKids/Music, including .flac when the browser supports it.
  • Artist and title are derived from folder and filename, then saved into RadioTrack.
  • Embedded artwork is extracted and stored in the shared radio artwork library for reuse across the site.
  • Successful tracks become available to the live player through the normal radio library flow.
  • Failed uploads are written to tmp/song_upload_failures_latest.json.
